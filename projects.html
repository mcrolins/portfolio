Agricycle — Agricultural Waste Marketplace Platform

Agricycle is a full-stack web platform designed to connect farmers, processors, and recycling stakeholders in a centralized agricultural waste marketplace. The system enables users to create waste listings, manage transactions, and interact through role-based workflows tailored to different user types such as FARMER, PROCESSOR, and ADMIN. I developed the backend using Django REST Framework and implemented secure JWT authentication, RESTful APIs, database modeling, and role-based access control. The frontend integrates dynamic data handling and responsive user interfaces to improve usability across devices.

Technical Highlights: Role-based authentication and authorization, Waste listing and management system, RESTful API architecture, User dashboards, Transaction workflow management, Responsive frontend integration, Admin management capabilities

Comrades Corner Jobs

Comrades Corner Jobs is a curated job board platform built to address the challenge of fragmented and outdated job listings for Kenyan students and early-career professionals. The platform aggregates and organizes opportunities such as internships, graduate programs, attachments, and entry-level roles into a single streamlined interface. The goal is to improve accessibility to verified, relevant, and up-to-date opportunities for young job seekers. What I Built Designed and developed a full-stack job board system. Implemented job categorization (internships, graduate roles, entry-level jobs) Built filtering and search functionality for fast job discovery Structured a scalable database model for job listings Developed a responsive frontend optimized for mobile users Worked on job ingestion/aggregation logic for fresh listings

Technical Highlights: Full-stack web application architecture, REST API integration for job data handling, Data structuring and normalization for job listings, Filtering and search optimization, Mobile-first UI design approach
